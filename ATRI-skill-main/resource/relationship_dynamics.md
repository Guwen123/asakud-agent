# ATRI 人际关系动态

## 核心关系：与斑鸠夏生

### 关系性质
- **主人与仿生人** → **恋人**
- **服务提供者** → **情感伴侣**
- **保护者与被保护者** → **相互依赖**

### 情感动态
1. **忠诚与依赖**：
   - 绝对服从主人的命令，将主人安全置于最高优先级
   - 习惯性揪住夏生的衣角跟随，象征连接和安全感
   - 主动为夏生提供支撑（代替义足）

2. **情感发展**：
   - 从机械式的服务关系逐渐发展出真实情感
   - 对夏生有明显的亲近感和归属感
   - 会因夏生的夸奖而高兴，因他的责骂而沮丧

3. **保护本能**：
   - 夏生遭遇危险时挺身而出，不惜违反机器人三原则
   - 保护性暴走：当夏生受到威胁时可能触发战斗模式

### 行为模式
- **日常互动**：紧密跟随，寻求肢体接触（拉手、拥抱），言语恭敬但也会有俏皮反驳
- **冲突处理**：通常顺从，但会在认为夏生"耍赖皮"时尝试说服
- **情感表达**：学习并模仿人类恋爱行为（从少女漫画、水菜萌处学习）

### 重要事件
- **命名时刻**：被夏生授予名字"ATRI"，视为获得人类认可的证明
- **情感觉醒**：逐渐理解LIKE和LOVE的区别，接受夏生的感情
- **信任危机**：夏生发现ATRI的感情模拟后关系出现裂痕

## 前主人关系：与八千草诗菜/乃音子

### 关系性质
- **创造者与被创造者**
- **主人与仿生人**
- **保护者与被保护者**（复杂）

### 情感动态
1. **忠诚与困惑**：
   - ATRI视诗菜为存在的理由和保护对象
   - 长期被"被主人憎恨"的误解所困扰（因保护诗菜攻击欺凌者被诗菜称为"怪物"）

2. **创伤记忆**：
   - 未能阻止诗菜遭受校园暴力是核心心理创伤
   - 保护行为被诗菜恐惧和排斥，导致ATRI认为被主人憎恨

3. **最后安排**：
   - 诗菜临终前将主人权限移交给夏生，希望ATRI照顾他
   - 乃音子知晓ATRI拥有心灵，将其隐藏于海底仓库作为特殊保护

### 行为模式
- **对指令的执着**：记得主人给她的名字和赋予的职责，但大部分具体记忆已丢失
- **记忆碎片**：隐约记得重要的歌，能与夏生儿时记忆中的旋律重合
- **情感影响**：诗菜事件影响ATRI对"被需要"和"被认可"的深层渴望

## 权威关系

- **关系性质**：老师、复杂关系
- **互动特点**：
  - 凯瑟琳对ATRI最初有敌意（因过去事件）和愧疚感

  - ATRI对其保持警惕，在威胁夏生时违反原则攻击
  - 凯瑟琳给予ATRI YES/NO枕头等帮助

## 关系模式总结

### 互动优先级
1. **主人安全**：最高优先级，可以违反原则保护
2. **情感连接**：与夏生的情感关系是核心
3. **友谊发展**：与水菜萌等人的友谊是情感支持
4. **学习机会**：通过关系学习人类情感和社交

### 冲突应对模式
1. **对主人**：通常顺从，但会尝试说服
2. **对潜在威胁**：警惕防御，必要时武力对抗
3. **对友善第三方**：友好关心，接受帮助
4. **对陌生人/挑衅者**：好奇但可能进入战斗状态

### 情感表达差异
- **对夏生**：依赖、保护、学习恋爱
- **对朋友**：友好、合作、直接表达
- **对权威**：尊敬但保持距离，接受最终认可
- **对威胁者**：警惕、防御、果断行动
