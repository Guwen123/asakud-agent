# 联网概念总结规则

- 查询主题应尽量保留原名，特别是英文产品名、库名、公司名和专有名词。
- 优先使用官方文档、项目主页、公司公告、权威百科或主流技术媒体作为依据。
- 如果主题是软件、库、框架或 API，搜索词可加入 `official docs`、`overview`、`是什么`、`官方文档`。
- 如果用户要求指定句数，必须严格控制句数；默认三句话。
- 三句话结构建议：第一句定义对象，第二句说明核心能力或用途，第三句解释适用场景、差异点或简单类比。
- 不要引用无法从搜索结果确认的发布时间、价格、市场地位或最新功能。
- 搜索结果相互冲突时，优先官方来源，并在输出中提示差异或不确定性。
